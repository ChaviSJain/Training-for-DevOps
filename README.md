# Training-for-DevOps

🚀 90-Day Full-Stack DevOps Training Project
Welcome to my training repository! Over 90 days, I've worked through a weekly curriculum combining hands-on coding, DevOps tools, containerization, automation, and deployment practices.

This README tracks what was learned and built during each week, along with related source code and links to documentation.

## 📅 Days 1–30: Foundations & Onboarding

### 🎯 Goals

- Understand the product: **Workspace ONE (WS1)**
- Get familiar with the team, tools, and development workflows
- Set up the dev environment
- Learn core concepts in:
  - Linux
  - GitHub
  - Ansible
  - CI/CD
  - Docker
  - AWS
  - Infrastructure as Code (IaC)
